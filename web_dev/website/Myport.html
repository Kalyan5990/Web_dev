Criminal Face Detection System

The Criminal Face Detection System application is built to detect the faces of criminals by matching them with the pre-existing data in the database. Although, there are so many ways to identify a criminal, this could be the best way, and building this project for advanced programmers is easy.

Abstract: The project is intended to use the images previously taken and identification will be done according to images taken of different people. This project aims to build an automated CFD system by leveraging the human ability to recall minute details in the face. Identification of criminals at the scene of a crime can be achieved in many ways like fingerprinting, DNA matching, or eyewitness accounts. Out of these methods, eyewitness accounts are preferred because they stand scrutiny in court and are a cost-effective method. It is possible that witnesses to a crime have seen the criminal, though in most cases, it may not be possible to completely see the face of the perpetrator.

Technologies Required:

TensorFlow, Core Java, Machine Learning, SQLite, OpenCV, Strong knowledge of advanced Java concepts
